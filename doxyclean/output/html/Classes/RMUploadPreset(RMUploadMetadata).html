serviceURL

This method constructs the NSURL from your Info.plist.

+ (NSURL *)serviceURL
Discussion

If sent to a preset whose bundle RMUploadPluginServiceHostKey is nil, it will throw an exception.

This will only be useful for your own web services where you can use the Info.plist preprocessor to switch between development and deployment servers.
